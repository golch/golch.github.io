Algospot 튜토리얼 DRAWRECT 문제

https://algospot.com/judge/problem/read/DRAWRECT

이번 문제는 직사각형의 3점의 좌표를 알 때, 나머지 한 좌표를 구하는 프로그램을 작성하는 문제다. 이걸 풀려고 종이에 사각형을 그리면서 중학교때 풀었던 수학문제였구나 라는 생각을 했다. 요즘은 초등학교에서 배우려나...?

풀이는 그림그리면서 생각이 났고, 일단 입력값을 받아서 처리해야 한다. 그런데 Scanner 의 next() 메소드를 써본적이 없어서 어떻게 받아오는지 잘 모르겠다. 구글링 구글링!ㅋㅋ

nextInt() 메소드를 사용해서 처리해야 공백이 있는 입력을 받아올 수 있군 ㅋㅋㅋ

아… 삽질했다 ㅋㅋ 직사각형이 좌표평면 위에 기울어져 있어도 구할 수 있는 프로그램을 짜고있었는데 문제를 다시 읽어보니 기울어질 수 없는 조건이네… ㅠㅠ

그럼 로직이 더 간단하지 ㅋㅋㅋ
아놔 튜토리얼이라고 해놓고 왜이렇게 어렵나 했네..

두 번이나 틀리고나서 결국 정답!ㅋㅋ

[code lang=text]
import java.util.Scanner;

public class Main {
public static void main(String[] args) {

Scanner sc = new Scanner(System.in);
int T = sc.nextInt(); //testcase 의 갯수.
int[] A = new int[T] ,B = new int[T];//구하고자 하는 좌표

for(int i=0; i<T; i++){

int x1,y1;
int x2,y2;
int x3,y3;//입력받은 좌표

x1 = sc.nextInt();
y1 = sc.nextInt();

x2 = sc.nextInt();
y2 = sc.nextInt();

x3 = sc.nextInt();
y3 = sc.nextInt();

if( (1 <= x1) && (x1 <= 1000)
&& (1 <= x2) && (x2 <= 1000)
&& (1 <= x3) && (x3 <= 1000)
&& (1 <= y1) && (y1 <= 1000)
&& (1 <= y2) && (y2 <= 1000)
&& (1 <= y3) && (y3 <= 1000)
){

if( ((x1 == x2) && (y1 == y2))
||((x2 == x3) && (y2 == y3))
||((x1 == x3) && (y1 == y3))
){

}else{
//핵심로직 : (A,B)를 구해야 함.
//(x1 - x2) = (A - x3)
//(y1 - y2) = (B - y3)

// A[i] = (Math.abs(x1 - x2) + x3);
// B[i] = (Math.abs(y1 - y2) + y3);

if(x1 == x2){
A[i] = x3;
}else if(x1 == x3){
A[i] = x2;
}else{
A[i] = x1;
}

if(y1 == y2){
B[i] = y3;
}else if(y1 == y3){
B[i] = y2;
}else{
B[i] = y1;
}
}

}

}

for(int i=0; i<T; i++){
if((1 <= A[i]) && (A[i] <= 1000)
&& (1 <= B[i]) && (B[i] <= 1000)){
System.out.print(A[i]);
System.out.print(" ");
System.out.print(B[i]);
System.out.println("");
}
}

}
}
