Algospot 튜토리얼 문제 ENCRYPT
이번에는 좀 쉽다.
문자열(String)의 짝/홀수를 구분해서 짝수 먼저 출력하고 뒤에 붙여서 홀수를 출력하면 된다.

https://algospot.com/judge/problem/read/ENCRYPT

이 문제에서는 문자열을 char로 잘라서 짝/홀수를 구분해서 다시 문자열로 담아주는 것이 핵심인듯 하다. 그리고 한 건이 아니고 여러 건을 실행해야 하니까 문자열의 배열(String[])을 사용해야 한다.
그 이외에 어려운 부분은 없었던듯.. 한글문제라서 헷갈리지도 않고..

 

 

[code lang=text]
import java.util.Scanner;

public class Main {
public static void main(String[] args) {

Scanner sc = new Scanner(System.in);
int T = sc.nextInt();//T는 1~10 사이의 정수
if((1<=T) && (T<=10)){
String cases = "";
char[] word = {};

String[] even = new String[100];
String[] odd = new String[100];

for(int j=0; j<T; j++){
cases = sc.next();
word = cases.toCharArray();

odd[j] = "";
even[j] = "";

for(int i=0; i<word.length; i++){
if((i % 2) == 0){//짝수이면
even[j] += word[i];
// System.out.println(even[j]);
}else{//홀수이면
odd[j] += word[i];
}

}
}

for(int j=0; j<T; j++){
System.out.println(even[j] + odd[j]);
}
}
}
}

[/code]
