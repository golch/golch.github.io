Algospot 튜토리얼 문제 HOTSUMMER

https://algospot.com/judge/problem/read/HOTSUMMER

이번엔 왠지 간단할 것 같다.
즉, 목표 전기 사용량을 입력받고 9시부터 18시까지의 사용량을 모두 더해서 목표보다 큰지 작은지를 판별하면 된다. 여기까지만 하고 튜토리얼은 졸업하고 다른 문제를 풀어봐야할듯ㅋㅋ

  1. 건물 개수는 T개. 제한은 없음
  2. 목표 전기 사용량을 입력값으로 받음.
  3. 9시~18시 까지 총 9개의 숫자를 입력값으로 받음. 공백으로 구분.
  4. 전력사용량은 모두 0~1000 사이의 정수.
  5. 전체 사용량의 합이 목표 이하이면 “YES”, 그렇지 못하면 “NO”를 출력한다.

[code lang=text]
import java.util.Scanner;

public class Main {
public static void main(String[] args) {

Scanner sc = new Scanner(System.in);
int T = sc.nextInt();

int[] electricGoal = new int[T];
int[] A09 = new int[T];
int[] A10 = new int[T];
int[] A11 = new int[T];
int[] A12 = new int[T];
int[] A13 = new int[T];
int[] A14 = new int[T];
int[] A15 = new int[T];
int[] A16 = new int[T];
int[] A17 = new int[T];

int[] sum = new int[T];

for(int i=0; i<T; i++){
electricGoal[i] = sc.nextInt();

A09[i] = sc.nextInt();
A10[i] = sc.nextInt();
A11[i] = sc.nextInt();
A12[i] = sc.nextInt();
A13[i] = sc.nextInt();
A14[i] = sc.nextInt();
A15[i] = sc.nextInt();
A16[i] = sc.nextInt();
A17[i] = sc.nextInt();

sum[i] = A09[i] +A10[i] +A11[i] +A12[i] +A13[i] +A14[i] +A15[i] +A16[i] +A17[i];

}

for(int i=0; i<T; i++){
if(electricGoal[i] >= sum[i]){
System.out.println("YES");
}else{
System.out.println("NO");
}
}
}
}
