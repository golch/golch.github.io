Algospot 튜토리얼 문제 URI
https://algospot.com/judge/problem/read/URI

왕초보급은 이제 졸업하고 다음 단계로 올라왔다. 빨리 어려운 것도 풀어보고 싶지만 일단은 차근차근...
이번 문제는 영어로 쓰여있어서 일단 문제 파악부터 해야한다.

음. 특수문자 %가 포함된 encoded string 을 진짜 특수문자로 치환하는 문제이다.
1. 테스트케이스 T는 1~100 사이의 정수이다.
2. 입력값(percent-encoded URI)의 길이는 최대 80 이다.
3. 입력되는 URI 를 decode 하시오.

음.. 잘 만든 것 같은데 런타임 오류 ㅠㅠ
폭풍 검색 해보니 java에서 String의 replace 메소드 사용할 때 ‘$’ 얘가 문제가 된다. ‘\$'로 처리해야 정상적으로 동작함.
이렇게 수정하고 다시 제출했는데 이번엔 오답!!!!

ㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋ
결국 범인은 ‘%’ 였다. 쉬운듯 안쉬운 문제 ㅠㅠ
예를 들어서 %2525 이런 입력이 들어왔을때 %25는 %로 치환되므로 %25 이렇게 출력되는 것이 맞다. 출제자의 의도는 한번만 치환과정을 거쳐야 함. 그런데 치환하는 코드를

[code lang=text]
uri[i] = uri[i].replaceAll("%25", "%");
uri[i] = uri[i].replaceAll("%20", " ");
uri[i] = uri[i].replaceAll("%21", "!");
uri[i] = uri[i].replaceAll("%24", "$");
[/code]

이렇게 짜면 %25 처리 후 다른 코드가 처리되어 두번 처리되는 경우가 있다. 결국 %25 치환코드를 맨 아랫줄로 내려서 해결!

3번이나 틀림ㅋㅋㅋㅋ

[code lang=text]
import java.util.Scanner;

public class Main {
public static void main(String[] args) {
Scanner sc = new Scanner(System.in);

int T = sc.nextInt();
String[] uri = new String[T];

if(( 1<=T ) && ( T<=100 )){//T는 1~100 사이의 정수.
for(int i=0; i<T; i++){
uri[i] = sc.next();

if((0 < uri[i].length()) && (uri[i].length() <= 80)){

uri[i] = uri[i].replaceAll("%20", " ");
uri[i] = uri[i].replaceAll("%21", "!");
uri[i] = uri[i].replaceAll("%24", "\\$");
uri[i] = uri[i].replaceAll("%28", "(");
uri[i] = uri[i].replaceAll("%29", ")");
uri[i] = uri[i].replaceAll("%2a", "*");
uri[i] = uri[i].replaceAll("%25", "%");
}

}

for(int i=0; i<T; i++){
System.out.println(uri[i]);
}
}
}
}

[/code]
